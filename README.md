# Quantum-algo-sdk

This is an SDK to help developers with basic quantum functions to build tools for the PQ world. Requires understanding of Linear Algebra, Calculus 1-3 and Differential Equations.

### Libs used

We use the following libs

- quantum tensors
- ts quantum
- q5m

`npm install @q5m/q5m quantum-tensors ts-quantum@0.9.0`
